# 第十八部分｜失败与流失案例库

## 57. FL-001｜未及时跟进，客户被其他中介成交上亿资产

### 事实

- 客户：互联网成长起来的网红型客户。
- 结果：因未及时跟进，客户被其他中介持续服务并购买上亿元资产。

### 根因

- 没有把长期维护放在第一优先级。
- 可能把“暂时不成交”误判为“低价值客户”。
- CRM 没有明确下一步动作和触达时间。

### 新规则

- 所有有效高净值客户必须设置 `next_action_date`。
- 客户暂不买，不等于停止服务。
- 每次触达要提供增量价值，不能只问“考虑得怎么样”。

### 训练能力

长期客户经营、机会识别、跟进纪律。

## 58. FL-002｜只在原有房源框架内找，错失二手成交

### 事实

- 客户：W 先生。
- 过程：多次带看仍未找到更适配房源。
- 结果：其他中介推荐二手房并成交。

### 根因

- 房源厚度不足。
- 思路可能局限于新房或现有资源。
- 没有从客户需求重新打开一二手全市场。

### 新规则

- 连续 3 次带看没有明显进展，必须重新诊断需求和扩大房源类型。
- 客户不应被顾问的新房/二手业务偏好限制。
- 建立板块全量房源地图和同业合作机制。

### 训练能力

房源全市场能力、需求重构、合作资源。

## 59. FL-003｜客户选择极高返佣中介

### 事实

- 客户：Z 先生。
- 结果：其他中介仅保留极少佣金，其余返给客户，客户选择对方。

### 根因

可能包括：

- 客户将服务视为同质化。
- 顾问价值没有在交易前被明确展示。
- 客户的首要决策标准本身就是现金返还。

### 新规则

- 前期就说明服务边界、佣金规则和价值。
- 形成可见的价值证明单，不等客户谈返佣时才解释。
- 不使用不合规或不可兑现的返佣承诺。
- 接受与价值观不匹配的合理流失。

### 训练能力

价值表达、客户筛选、佣金透明、边界管理。
