## 54. MG-001｜长期找房后实现别墅半定制交付

### 内部事实

- 客户：Y 先生，高净值家庭。
- 需求：购买别墅，偶尔回成都居住。
- 预算：3500 万元以上。
- 最终项目：交子缦华别墅。
- 决策周期：约一年半。
- 最大阻力：同时满足别墅形态、精装、地段、6 房以上的房源长期缺失。

### 马老师关键动作

1. 持续维护长周期需求，没有用不合适房源逼单。
2. 重新梳理客户真正不可妥协的功能和地段条件。
3. 匹配到接近条件的清水房源。
4. 与开发商进行深度斡旋，把清水方案推动为面向客户的半定制精装交付。
5. 同时争取较好的价格和交易条件。

### 结果

客户最终购买，并认可马老师能够解决决策过程中的实质性问题。

### 可复制方法

- 当市场没有“现成标准答案”时，尝试重构供给，而不是持续无效找房。
- 长周期客户需要持续跟进和需求不变项管理。
- 开发商谈判不只谈价格，还可谈产品、交付和定制。

### 对外内容标题示例

> “看了一年半都没有合适别墅，我们最后不是找到标准答案，而是把交付方案重新谈了一遍。”

### 公开边界

隐藏客户身份、企业、具体付款和特殊合同条款；项目名称与金额是否公开需取得授权。

## 55. MG-002｜在高返佣竞争中以产品和交付赢得信任

### 内部事实

- 客户：L 先生。
- 需求：大平层改善自住。
- 初始预算：3000 万元以内。
- 最终项目：蔚蓝卡地亚·天览整层。
- 最大阻力：房源选择复杂；外部中介用高返佣撬单；高佣金优惠与项目交付确定性之间存在冲突。

### 马老师关键动作

1. 把客户注意力从“返多少”拉回到“买到什么、能否交付”。
2. 系统比较产品力、交付能力、空间尺度和总成本。
3. 推荐在产品、面积和交付上更适配、但渠道佣金并不占优的方案。
4. 为客户争取较好的成交价格。
5. 持续用透明和实际解决问题维护信任。

### 结果

客户最终选择蔚蓝卡地亚·天览整层，未要求马老师返佣。

### 可复制方法

- 返佣竞争不是口头反驳，而是用产品、交付、价格和风险的完整证据重建价值标准。
- 佣金高低不能替代客户资产适配。
- 当客户看见顾问站在其利益一侧，价格型竞争会弱化。

### 对外内容标题示例

> “别人愿意返更多，他为什么最后选了佣金并不高的项目？”

### 公开边界

不得披露具体佣金比例、同行身份和客户隐私；内容重点放在决策逻辑，不攻击同行。

## 56. MG-003｜稀缺二手房源与复杂后端促成成交

### 内部事实

- 客户：Z 先生。
- 需求：购买大平层。
- 预算：约 3500 万元。
- 最终房源：麓湖云龙阁稀缺次顶层约 370㎡户型。
- 成交价：约 3000 万元。
- 最大阻力：长期没有满意房源；外部中介持续带看其他项目；交易后端存在他人无法解决的复杂问题。

### 马老师关键动作

1. 通过内部资源获得稀缺房源。
2. 用客户真实需求快速确认房源适配。
3. 管理竞品干扰和客户决策节奏。
4. 协调并解决交易后端复杂问题。

### 结果

客户完成购买。团队提供信息称该交易刷新当时麓湖同类大平层清水成交单价纪录，**此表述对外使用前必须核验时间范围、样本范围、清水/精装口径和交易凭证**。

### 可复制方法

- 稀缺房源能力必须与交易落地能力同时存在。
- 二手豪宅的核心壁垒通常在“房源 + 后端”，而非单一带看。
- 纪录不是内容重点，解决复杂问题才是品牌资产。

### 对外内容标题示例

> “稀缺房源拿到了，还不等于能成交：真正难的是后端。”
