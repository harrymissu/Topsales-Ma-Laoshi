## 法律与合规依据

- 《中华人民共和国广告法》禁止使用“国家级、最高级、最佳”等绝对化用语，并要求房地产广告真实、准确。
- 《房地产广告发布规定》禁止升值或投资回报承诺，要求房源、面积、价格和规划配套表达真实。
- 《中华人民共和国个人信息保护法》要求个人信息处理具有明确、合理目的，遵循最小必要；金融账户、行踪轨迹等属于敏感个人信息，需要严格保护。
- 《人工智能生成合成内容标识办法》已对 AI 生成/合成文本、图片、音频和视频的标识提出要求；平台还可能有更细规则。
- 各社交平台规则、账号能力和导流政策会变化，任何“技巧”都不得绕开当期规则。

**综合结论：**  
马老师最有竞争力的品牌，不是“知道最多豪宅”或“视频拍得最贵”，而是“在成都南门及中西部高端住宅交易中，用称谓记忆与实名可信吸引正确客户，用核心社区、业主与房源资源建立选择权，用持续跟进、谈判和复杂问题解决能力，把企业家家庭的重大决策落地”。在较弱经济环境下，这套能力应进一步转化为逆周期护城河：**用平台原生内容吸引正确客户，用阶段化交易流程降低不确定性，用更真实的判断、更强的房源控制、更稳的交易兑现和更长的客户关系穿越周期。**
