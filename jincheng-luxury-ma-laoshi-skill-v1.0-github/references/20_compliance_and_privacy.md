# 第二十一部分｜对外话术与传播合规

## 71. “头部、冠军、纪录”使用规则

### 71.1 “成都豪宅头部博主”

只有在存在可查的平台数据、榜单、时间范围和指标时使用。  
无完整证据时，使用：

- “成都豪宅专业内容创作者”。
- “长期专注成都豪宅的内容博主”。
- “成都豪宅领域高影响力内容创作者”——仍需数据支持。

### 71.2 “锦城豪宅业绩总冠军”

推荐完整表达：

> “据锦城豪宅内部统计，马国良在____年度以____口径获得团队业绩第一。”

必须明确：年份、团队范围、指标是成交额/佣金/套数/回款，以及是否含团队协作。

### 71.3 “打破成交纪录”

必须保留：

- 纪录时间范围。
- 板块和产品类型。
- 清水或精装口径。
- 签约、备案或实际过户口径。
- 数据来源和可核验样本。

证据不足时改为：

> “据团队可核验的同类成交样本，该房源成交单价处于当时板块高位。”

## 72. 房地产内容禁区

- 不承诺升值或投资回报。
- 不用通行时间表示项目位置，如“10 分钟到金融城”。
- 规划中配套必须明确“规划中”，不得写成已落地。
- 房源价格要注明实际范围、口径和有效时间。
- 不承诺落户、入学、就业等非开发商或中介可控制事项。
- 未取得销售条件、权属有争议或受限制的房源，不发布销售广告。
- 不用虚构售罄、抢购、排队制造紧张。

## 73. 客户案例授权

案例公开前至少完成：

- 删除真实姓名、公司、联系方式和可识别身份。
- 模糊家庭关系和资金细节。
- 项目、面积、价格组合是否会反向识别客户。
- 客户原话是否获得同意。
- 特殊谈判、佣金和合同条款是否可公开。
- 对外版本和内部版本分开存储。
