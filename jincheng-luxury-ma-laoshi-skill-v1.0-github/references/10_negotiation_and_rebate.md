# 第十五部分｜异议、谈判与反佣竞争

## 46. 异议分类

| 异议 | 表面表达 | 真实可能原因 |
|---|---|---|
| 价格 | “有点贵” | 没有形成价值参照，或担心买贵 |
| 房源 | “再看看” | 核心需求未被满足，或房源厚度不足 |
| 交付 | “怕做不出来” | 对开发商、精装、定制或工期不信任 |
| 家庭 | “回去商量” | 决策人未参与，或存在未表达的否决 |
| 时间 | “不急” | 没有购买必要性，或资金时间不匹配 |
| 返佣 | “别人返得更多” | 客户把服务同质化，价值没有被看见 |
| 顾问信任 | “我再问问” | 证据不足、信息不透明或关系未建立 |

## 47. 谈判准备清单

- 买方最高可接受总成本。
- 买方最重要的非价格条件。
- 卖方/开发商真实动机和决策权限。
- 双方时间压力。
- 可交换筹码。
- 每一步让步需要换回什么。
- 最佳替代方案 `BATNA`。
- 谈判失败后的退路。

## 48. 让步矩阵

| 可谈条件 | 买方价值 | 对方成本 | 优先级 |
|---|---:|---:|---:|
| 总价 | 高 | 高 | 视证据使用 |
| 付款节奏 | 高 | 中 | 高 |
| 精装/定制 | 高 | 中高 | 高 |
| 车位/储藏室 | 中高 | 中 | 中高 |
| 家具家电 | 中 | 低中 | 中 |
| 物业费/服务 | 中 | 中 | 中 |
| 交付时间 | 高 | 视情况 | 高 |
| 税费承担 | 高 | 高 | 二手重点 |
| 腾退和租约 | 高 | 中 | 二手重点 |

## 49. 反佣价值防守

原则：**透明、合规、不过度卷、用实际结果证明服务价值。**

### 49.1 必须做到

- 如实说明佣金或服务费的来源与规则。
- 不私自承诺公司或项目不允许的返佣。
- 不攻击、诋毁其他中介。
- 不用“客户有钱不在乎”作为理由。
- 把价格争取、风险避免、交付确定性和时间成本具体化。

### 49.2 价值证明单

与客户沟通时，可列出：

- 拿到的稀缺或非公开房源。
- 谈下的价格和非价格条件。
- 识别并关闭的重大风险。
- 为客户节省的时间和无效带看。
- 解决的产权、付款、定制、交付或家庭问题。
- 交易后仍需承担的服务责任。

### 49.3 建议表达

> “返佣是一个清晰的价格问题，您完全可以比较。但这笔交易更大的成本，是买错房、买贵、交付不确定或后端无法落地。我会把我的服务、能争取的条件和风险处理写清楚，您再决定这份价值是否值得。”

### 49.4 接受合理流失

若客户只以返佣最大化为唯一标准，且不认可专业价值，可以尊重选择，不通过无底线承诺换取成交。
