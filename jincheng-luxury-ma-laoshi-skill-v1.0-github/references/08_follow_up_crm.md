# 第十部分｜高净值客户跟进系统

## 29. 跟进第一原则

高端客户不是靠高频骚扰成交，而是靠“持续存在 + 每次有增量价值”。

每个有效客户必须有：

- 当前阶段。
- 未解决的第一问题。
- 下一步动作。
- 下一次触达日期。
- 触达价值。
- 流失风险。

`next_action_date` 为空，视为 CRM 不合格。

## 30. 建议触达节奏

| 客户状态 | 节奏 | 可提供的价值 |
|---|---|---|
| 明确目标、近期成交 | 24-72 小时 | 新房源、谈判进展、风险核验、家庭共识 |
| 有需求、等合适房源 | 每 7-14 天 | 精准房源、板块变化、成交案例、价格变化 |
| 长周期客户 | 每月或重大节点 | 市场简报、稀缺房源、政策和项目进展 |
| 暂不适合买 | 低频季度维护 | 明确告诉客户等待条件，不催促 |
| 已成交客户 | 交付节点 + 半年/年度 | 验房、装修、物业、资产档案、转介绍服务 |

频率必须根据客户偏好调整，不能机械群发。

## 31. 失联和流失预警

出现以下情况，标记为高风险：

- 连续两次触达没有新价值。
- 客户频繁接受其他中介带看。
- 客户开始只讨论返佣，不讨论房源适配。
- 家庭决策人从未出现。
- 客户看房超过 5 次但需求没有更新。
- 关键房源释放后未在约定时间通知。
- 顾问没有明确下一步。

处理方式不是“催”，而是重新诊断：客户究竟缺房源、缺确定性、缺家庭共识，还是已经不再需要服务。
