# 第十六部分｜复杂交易与极致交付

## 50. Deal War Room

每笔复杂交易建立作战室：

| 角色 | 责任 |
|---|---|
| 首席顾问 | 客户关系、决策、谈判、总协调 |
| 房源负责人 | 开发商/业主、房源与条件核验 |
| 交易负责人 | 合同、付款、过户、交付节点 |
| 法律/产权专业方 | 产权、合同、争议和合规意见 |
| 银行/资金方 | 贷款、解押、资金时间 |
| 税务/评估方 | 税费、评估和专业核算 |
| 验房/设计/工程 | 房屋质量、精装、定制与交付 |
| 内容与隐私负责人 | 案例授权、脱敏和传播边界 |

## 51. 交易风险六类

1. **主体风险**：产权人、共有、授权、公司主体。
2. **资产风险**：质量、噪音、景观、设备、物业、租约。
3. **资金风险**：贷款、解押、资金监管、付款时间。
4. **合同风险**：定金、违约、交付、变更、家具清单。
5. **家庭风险**：否决、情绪、资产安排、使用冲突。
6. **信息风险**：隐私、客户身份、交易价格和传播。

## 52. 风险登记表

```markdown
| 风险 | 影响 | 概率 | 证据 | 解决动作 | 责任人 | 截止时间 | 状态 |
|---|---|---|---|---|---|---|---|
| 精装定制未写入合同 | 高 | 中 | 仅有口头承诺 | 形成材料与节点附件 | 顾问/开发商 | 签约前 | 处理中 |
```

## 53. 定制交付专项规则

涉及清水改精装、半定制、材料升级或格局调整时，必须明确：

- 设计图和改造范围。
- 材料、品牌、型号和替代规则。
- 样板、验收标准和质量责任。
- 费用和付款节点。
- 交付日期和延期责任。
- 变更审批流程。
- 物业、消防、结构和报建限制。

未经书面确认，不得把定制能力作为确定卖点。
