# 16d｜平台资质与房产类目准入（发布前置检查）

> 本文件属于 jincheng-luxury-ma-laoshi Skill v1.0 的引用文件。
> **定位：资质合规是 0，创作技巧是后面的 1。** 账号资质不合规时，16a 的一切创作建议暂停执行。
> 平台规则会变化（as_of 2026-07-12）。本文件只固化"必须核验哪些项"，具体要求以各平台创作者中心 / 规则中心当期页面为准；AI 输出涉及资质结论时一律标注"以当期官方要求为准"。

## Q-1. 为什么单列本模块

房产属于各平台的强监管垂类。资质问题的处罚不是"限流一条"，而是断私信、封类目、封号，且连带矩阵账号。任何新账号启动、老账号转型、直播开播、房源类内容发布前，先过本文件的检查清单。

## Q-2. 通用资质检查清单（所有平台）

发布房源类、成交类、市场判断类内容前，逐项确认：

1. **主体身份**：账号主体是个人、个体户还是公司（锦城豪宅）？发布房源信息的账号是否需要挂靠机构资质？
2. **从业资质**：马老师及团队成员的房地产经纪从业身份、机构备案是否可提供给平台审核？
3. **类目准入**：该平台房产垂类是否需要单独申请/报白/认证？未准入前哪些内容形态被禁止（挂房源组件、直播讲盘、私信留资）？
4. **房源信息合规**：房源发布是否要求真实房源核验码、委托证明、价格有效期？
5. **广告标识**：与开发商合作的内容是否构成商业推广，需要平台商单/广告标识？
6. **AI 内容标识**：AI 生成或合成素材是否已按当期法律和平台要求标识？

任何一项"不确定"，输出中先给核验动作，再给内容方案。

## Q-3. 分平台资质要点（须按当期官方核验）

| 平台 | 核验要点（不是结论，是待查项） | 未达标时的替代动作 |
|---|---|---|
| 抖音 | 房产垂类经纪人身份认证要求；机构蓝 V 与员工号绑定规则；房源类组件/团购/线索工具的开通条件；直播讲盘的类目权限 | 只发不含具体房源与价格承诺的板块教育、避坑类内容；不开房源直播 |
| 小红书 | 专业号认证与行业类目；房产类目笔记与留资组件的准入；蒲公英商单对房产内容的要求 | 以家庭场景、决策清单类内容为主，不挂房源、不引导站外 |
| 微信视频号 | 房产类目直播与橱窗限制；企业认证与小程序（锦城豪宅小程序）挂载条件 | 视频号导公众号深度文章，转化走企业微信 |
| 微信公众号 | 房源信息发布的主体资质要求；金融、投资回报类表述的红线 | 只做板块档案、政策解读、方法论内容 |
| 快手 | 房产垂类（理想家等）准入与经纪人认证；本地直播规范 | 同抖音替代策略 |

## Q-4. 账号主体选择建议

- **马老师个人 IP 号**：个人认证 + 机构关联，承载判断与方法内容；避免让个人号成为唯一的房源发布口，降低单点封禁风险。
- **锦城豪宅机构号**：承载房源、直播、组件与留资工具，资质文件齐备后优先在机构号开通垂类权限。
- **团队成员号（矩阵）**：统一以员工号身份绑定机构，禁止用无资质个人号发布具体房源与价格。

## Q-5. 违规分级与响应

- **红色（立即停）**：无资质发布房源/价格承诺、直播中承诺升值学区贷款结果、诱导站外交易 → 立即下架，按 SL-6 舆情流程处理，复盘进 `data/`。
- **黄色（限期改）**：缺广告标识、缺 AI 标识、房源信息过期 → 48 小时内补正或下架。
- **绿色**：资质齐备、信息带日期、留资走站内工具。

## Q-6. AI 输出规则

1. 用户要求生成任何房源类内容时，先问/先查该账号的资质状态；未知则在输出头部加"资质前置检查"清单。
2. 不得建议以个人生活号"绕开"垂类资质发布房源。
3. 平台准入要求本身也是时效信息：引用时标日期，超过 90 天提示重查。
