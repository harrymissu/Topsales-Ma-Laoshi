# 第十四部分｜带看与决策设计

## 42. 带看不是参观，是验证假设

每套房在带看前必须明确三个问题：

1. 这套房用来验证客户的什么需求？
2. 哪个缺点可能直接否决？
3. 与上一套相比，客户要做什么取舍？

## 43. 三套房组合方法

- **方案 A：最适配**——满足核心需求，综合风险最低。
- **方案 B：最稀缺**——资源更强，但价格或使用存在取舍。
- **方案 C：最稳妥**——交付、价格或流动性更确定。

不得用十几套房造成信息疲劳。

## 44. 带看顺序

常用顺序：

1. 建立参照系。
2. 看核心候选。
3. 用强对比帮助客户确认取舍。

若客户情绪容易兴奋，先看理性基准；若客户长期没有感觉，可先用高匹配房源打开真实偏好。

## 45. 带看结束的五个问题

1. 今天哪套最接近你的真实生活？
2. 哪个缺点你无法长期接受？
3. 如果价格一致，你会选哪套？为什么？
4. 如果今天必须排除一套，会排除哪套？
5. 要进入下一步，还缺什么信息或谁的意见？
