# 第二十部分｜个人 IP 与营销传播引擎

## 62. 内容目标

内容不是追求最大播放量，而是实现：

- 让目标客户知道马老师能解决什么问题。
- 让客户在见面前完成初步信任。
- 筛掉只看返佣、不认可专业价值的低适配线索。
- 让业主和开发商愿意提供稀缺资源。
- 用案例反哺销售和团队训练。

## 63. 内容漏斗

```text
市场观点/板块判断
→ 真实与反常识内容
→ 复杂案例与问题解决
→ 私域咨询
→ 需求诊断
→ 精准带看
→ 成交与交付
→ 脱敏案例
→ 老客转介绍
```

## 64. 八大内容支柱

1. **成都南门真实判断**：高新区、麓湖、天西及核心项目分化。
2. **一套房为什么不值得买**：敢于公开讲缺点和不适配人群。
3. **企业家买房决策**：时间、隐私、家庭、企业现金流与社交需求。
4. **一手时机、二手时间**：用真实案例讲两套逻辑。
5. **豪宅非标交易**：产权、定制、交付、谈判和复杂后端。
6. **成交复盘**：重点讲解决了什么，不炫客户和总价。
7. **业主卖房策略**：定价、筛客、保密、议价和周期。
8. **四川及中西部城市比较**：客户应该在成都、家乡还是其他核心城市购买。
9. **弱周期买卖决策**：买、卖、换、等、叫停，以及现金流、时间成本和真实折价。
10. **核心社区与业主资源**：库存、成交、降价、非公开房源和隐私委托。

## 65. 内容价值比例

建议：

- 35% 判断、教育与弱周期决策。
- 25% 案例和问题解决。
- 15% 核心社区、板块与项目。
- 15% 业主服务、二手库存和出售策略。
- 10% 人物、团队和价值观。

避免内容长期变成单一探盘或豪宅炫耀。

## 66. 标准内容结构

```text
1. 冲突：客户为什么看了很久仍不买？
2. 事实：预算、用途、必须项和市场现实。
3. 判断：真正的问题不是表面问题。
4. 证据：房源、成交、产品、交付或风险。
5. 取舍：这套房适合谁，不适合谁。
6. 结论：买、等、换方向或叫停。
7. 低压力 CTA：先做判断，不急着看房。
```

## 68. 个人品牌金句

可作为内容结尾或品牌签名，避免每条都重复：

- “我不负责把每套房都说成好房，只负责把适合和不适合讲清楚。”
- “豪宅交易的价值，不是多带几套，而是把真正的问题解决掉。”
- “一手看时机，二手看时间，最终看的是和家庭是否适配。”
- “客户兴奋的时候，我更有责任把缺点讲清楚。”
- “赚钱是解决问题后的自然结果，不是每一单的起点。”
- “卖房卖的不是嘴，是资源、判断和兑现。”

## 69. CTA 标准

推荐低压 CTA：

> “把预算、用途、时间和三个必须项发给我，先判断现在该不该买、该看哪一类，不急着安排带看。”

避免：

- “最后机会，马上私信。”
- “错过再无。”
- “闭眼买、稳赚、必涨。”

## 70. 内容效果指标

不只看播放量，重点看：

- A8+ 有效咨询数。
- 有效咨询率。
- 内容到深度诊断转化率。
- 诊断到带看转化率。
- 内容辅助成交金额。
- 业主委托和稀缺房源线索。
- 老客转介绍。
- 负面反馈、隐私和合规事件为零。

# 第二十部分C｜融合后的全域内容与成交系统

## OM-1. 总体原则

```text
公域不是成交场，而是问题发现与信任筛选场；
私域不是催单场，而是深度诊断与共识建立场；
带看不是参观场，而是验证假设场；
成交不是内容终点，而是案例、复购与业主资源的新起点。
```

## OM-3. 三层内容漏斗

### A. 触达层：20%

目标是让正确的人停下来，不追求泛流量最大化。

- 反常识判断。
- 预算决赛局。
- 城市和板块分化。
- 典型决策冲突。
- 高审美产品和空间故事。

### B. 信任层：55%

这是马老师的核心内容。

- 真实成功和失败案例。
- 为什么叫停客户。
- 一手和二手如何比较。
- 复杂交易如何关闭。
- 房源和价格证据。
- 企业家家庭决策。
- 业主委托与隐私保护。

### C. 转化层：25%

- 当周房源池和买方需求。
- 20 分钟诊断入口。
- 业主委托说明。
- 私享看房和板块小型沙龙。
- 市场报告、决策表和卖房手册领取。

## OM-4. 马老师十二大栏目

1. **《马老师先劝你别买》**：明确不买、等、换方向的条件。
2. **《3000 万决赛局》**：同预算三种配置逻辑，而不是简单排名。
3. **《成都南门房源真相》**：活跃供应、成交、稀缺和假稀缺。
4. **《麓湖库存周报》**：项目、组团、产品和价格变化；公开版不暴露非公开房号。
5. **《一手看时机》**：供应、加推、交付、价格和开发商谈判窗口。
6. **《二手看时间》**：业主动机、挂牌周期、替代房源和议价空间。
7. **《别人解决不了的后端》**：合同、产权、交付、定制、资金和家庭共识。
8. **《企业家家庭决策图》**：本人、配偶、父母、孩子和财务安排如何共识。
9. **《业主为什么把钥匙交给我》**：筛客、保密、定价和反馈机制。
10. **《四川老板到成都买什么》**：绵阳、宜宾、德阳、泸州等客源城市与成都落脚需求。
11. **《豪宅交付之后》**：验房、设计、出租、物业和二次出售。
12. **《马老师复盘错过的客户》**：公开失单教训，建立真实与学习能力。

## OM-5. 内容选题四象限

|  | 高意图 | 低意图 |
|---|---|---|
| 高专业 | 决策、案例、房源、交易、业主委托 | 城市、建筑、设计、生活方式 |
| 低专业 | 价格速报、活动通知 | 猎奇豪宅、泛情绪、名人和炫富 |

资源分配规则：

- 高意图 + 高专业：优先，占创作资源 50% 以上。

- 高意图 + 低专业：只能作为时效补充，必须核验。
- 低意图 + 低专业：原则上不做。

## OM-8. 标题公式

### 决策型

```text
预算 + 冲突 + 判断边界
```

示例：

- “3000 万以内，为什么我最后建议客户买佣金最低的那一套？”
- “客户看了一年半别墅，我没有继续找，而是把交付方式改了。”
- “麓湖 370㎡次顶为什么值得谈，为什么也有人不该买？”

### 业主型

```text
业主担忧 + 服务动作 + 可验证结果
```

示例：

- “豪宅业主最怕的不是卖慢，是被错误曝光。”
- “一套房交钥匙前，我要先和业主确认这 12 件事。”
- “挂牌三个月没结果，先别急着降价，先看买方池是否真实。”

### 反常识型

```text
常见兴奋点 + 冷静检查
```

示例：

- “样板间越惊艳，我越要提醒客户看这四个交付风险。”
- “返佣最多，不等于你的总成本最低。”

## OM-9. “讲缺点”五步法

```text
1. 先讲事实，不用情绪词。
2. 说明缺点影响哪类生活场景。
3. 判断是否可修复、可谈判或只能接受。
4. 给出替代项目或产品。
5. 明确适合谁、不适合谁。
```

禁止为了“敢说真话”而故意攻击项目、开发商、同行或业主。

## OM-10. 内容证据三角

每个重要观点必须尽量同时有：

1. **市场证据**：供应、成交、挂牌、产品或交付事实。
2. **案例证据**：真实客户如何决策，内部版可完整，公开版脱敏。
3. **边界证据**：什么条件变化后结论会失效。

示例：

> “这套房当前相对有议价条件”不是结论终点，还要说明业主动机是否核验、同类替代有多少、价格有效至何日，以及哪些付款条件会改变谈判空间。

## OM-11. 公域到成交的八步链路

```text
1. 内容触达
2. 关键词或私信进入
3. 线索来源编码
4. 5 分钟初筛
5. 20 分钟深度诊断
6. 一页 Client Memo
7. 三套以内方案与带看
8. 成交 / 等待 / 叫停 / 转长期维护
```

### 初筛五问

1. 购买或出售的核心原因是什么？
2. 预算、资金时间和是否需要先卖后买？
3. 谁使用、谁拍板、谁可能反对？
4. 看过哪些项目，为什么没有结果？
5. 希望在什么时间出现什么结果？

### 响应 SLA

| 线索 | 内部响应 | 客户触达原则 |
|---|---|---|
| A：明确预算、项目、30 天内行动 | 工作时间 30 分钟内进入责任人队列 | 先确认方便沟通时间，不连续催促 |
| B：明确改善需求、1—6 个月 | 4 小时内分配 | 提供一项增量信息，再预约诊断 |
| C：研究期或长期关注 | 24 小时内归档 | 进入内容培育，不频繁私聊 |
| 业主直委 / 稀缺房源 | 30 分钟内分配高级顾问 | 先谈保密、核验和看房规则，不先承诺价格 |

内部响应快不等于高压销售；高端客户更看重节奏、价值和边界。

## OM-13. 内容 KPI

### 业务 KPI

- 有效买方咨询数。
- 有效业主委托数。
- A/B 级线索比例。
- 诊断完成率。
- 诊断到带看率。
- 带看到明确意向率。
- 内容辅助成交金额与套数。
- 非公开房源获取数。
- 老客转介绍和复购。

### 品牌 KPI

- “马老师 + 马国良 + 锦城豪宅”跨平台搜索一致性。
- 核心栏目连续更新率。
- 长文、短视频和私域之间的互相引用。
- 客户主动引用某条内容的比例。
- 业主因内容主动委托的比例。

### 风险 KPI

- 信息错误更正次数。
- 房源过期仍发布次数。
- 客户隐私事件。
- 夸大、收益承诺和虚假稀缺事件。
- 多账号重复触达和内部撞单。

## OM-14. 30 / 60 / 90 天执行计划

### 前 30 天：统一身份与证据

1. 所有平台统一为“马老师｜锦城豪宅”，简介首句绑定实名“马国良”。
2. 完成三条置顶视频和一页公开履历。
3. 将 3 个成功案例、3 个失单案例分别做成短视频、视频号和公众号版本。
4. 建立内容 ID、线索来源 ID 和房源 ID。
5. 对现有内容中的“天花板、捡漏、必看、冠军、头部”做一次证据审计。
6. 建立 20 个南门核心项目的最小卡片。

### 31—60 天：形成栏目与业主端入口
