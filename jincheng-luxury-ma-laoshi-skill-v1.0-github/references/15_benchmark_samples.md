# 第二十部分B｜中国豪宅经纪与内容样本拆解

## BM-1. 样本选择说明

以下账号和人物是基于中国社交媒体与公开媒体可检索度选择的“方法样本”，不是客观排名。重点拆解其可观察的内容、定位、房源与转化方法；涉及成交、粉丝和房源量的数据，若来自账号简介或采访，均视为公开自述。

## BM-2. 代表性样本与可迁移能力

| 样本 | 公开可见方法 | 值得吸收 | 不应照搬 |
|---|---|---|---|
| 样本A（上海·业主圈型） | 账号简介突出多年豪宅服务、企业家客户和隐私；媒体采访描述其长期经营业主与买方资源、测试多账号、持续高频创作 | 把“业主圈 + 买方圈 + 自有房源”做成业务底盘；账号实验要有数据和复盘；公开承诺隐私保护 | 不能把采访中的房源量、收入或成交直接当独立事实；高频不等于高质量 |
| 样本B（上海·极致聚焦型） | 媒体采访称其将主营范围收缩到 5 个小区，围绕清晰女性/家庭决策者画像持续发布大量小红书笔记 | 极致聚焦少量社区；内容围绕真实买方画像；搜索占位和长期累计 | 不照搬机械刷量；不能牺牲内容准确性和客户体验 |
| 样本C（上海·业主交钥匙型） | 公开视频突出业主交钥匙、团队在核心社区的“战绩”、持续讲翠湖等标杆项目 | 业主信任和钥匙权是强证据；深耕核心小区；公开团队成交能力时讲口径 | 不公开可识别业主隐私；“战绩”必须可核验，不用夸张炫耀 |
| 样本D（品牌矩阵型） | 可见多个带“样本D品牌”的个人账号，内容兼具房源、团队文化、职业与生活方式；年度品牌视频获得较高公开互动 | 专家矩阵而非单一网红；组织文化可吸引业主、客户和人才；多账号分工 | 不让矩阵变成重复搬运；个人生活内容不应压过专业判断 |
| 样本E（成都·编号连载型） | 公开内容长期编号连载，围绕成都各板块和具体项目做 7—8 分钟深度测评，形成可检索目录 | 系列编号、项目档案和本地长期覆盖；让客户可连续追看 | “必看、急售、性价比”等措辞要有证据和时效；测评不能只做卖点 |
| 小春哥｜锦城豪宅 | 知乎长文建立 100 分、8 维度买房评价；公开指南超过 2 万字，并做成都板块系统教育 | 长内容成为搜索资产；用统一模型解释复杂市场；从城市到板块再到项目 | 评分不能替代客户适配；涉及涨跌与投资判断必须标日期和证据 |
| TONY｜锦城豪宅 | 公开视频重视大宅空间、设计和沉浸式体验，并有跨城市顶豪内容 | 视觉标准、空间动线、建筑和设计语言；用外部优秀产品提高成都判断尺度 | 视觉震撼不能等同可买；必须补交易、价格、交付和适配判断 |
| 样本F（空间叙事型）等内容型经纪/创作者 | 公开账号以全球豪宅、人物、空间和生活场景组织视频，弱化传统房源口播 | 把户型变成生活故事；提升审美、镜头和叙事；扩大非即时买房受众 | 内容影响力不等于经纪成交能力；不能照搬纯猎奇或窥私路线 |

## BM-3. 十项可复制能力

### BM-3.1 极致细分，而不是泛豪宅

学习 样本B 的核心不是“发 2000 篇”，而是把范围缩小到自己最有资源和判断力的少量社区。

马老师建议建立三层主场：

```text
第一层：高新区、麓湖、天西
第二层：金融城—大源—锦城湖—麓湖的企业家改善链
第三层：20—30 个核心项目 + 5—8 个必须赢下的标杆社区
```

每个核心社区至少做到：

- 了解全部产品线和户型。
- 维护活跃挂牌、历史成交和业主动机。
- 有至少 3 个真实优缺点判断。
- 有替代项目和跨板块比较。
- 有业主触达和买方池。
- 有一篇长文、一组短视频和一张内部房源地图。

### BM-3.2 经营业主圈，而不只经营买方流量

豪宅经纪最稀缺的不是“看房客户”，而是：

- 业主直接委托。
- 非公开出售意愿。
- 可控的看房权和钥匙。
- 对价格、家具、付款和交付的真实沟通权。

内容必须同时对买方和业主说话。

### BM-3.3 用系列化形成“可检索知识库”

借鉴样本E的编号连载和小春哥的系统长文，但把马老师的系列聚焦到解决问题：

- 《马老师叫停清单》第 001—100 期。
- 《成都南门核心项目档案》第 001—050 期。
- 《豪宅复杂交易复盘》第 001—030 期。
- 《业主卖房周报》第 001—052 期。
- 《一手时机 / 二手时间》第 001—040 期。

编号不是形式，而是让客户知道这是可持续更新的专业资产。

### BM-3.4 明确内容面对谁

一条内容只能有一个主要决策人：

- 企业家本人：效率、资产、隐私、时间。
- 配偶或家庭主要居住者：空间、审美、孩子、日常体验。
- 家庭财务决策者：资金安排、风险、退出和税费。
- 卖方业主：价格、周期、筛客、保密和执行。
- 外地四川企业家：成都与家乡、通勤、家族和长期落脚。

不再使用“各位老板都适合”的泛化表达。

### BM-3.5 视觉能力服务判断

高品质画面应回答：

- 这个空间如何被使用？
- 真实尺度如何？
- 动线有什么代价？
- 景观在不同时间如何变化？
- 装修是否遮盖产品硬伤？
- 维护成本和隐私成本是什么？

只拍“豪、贵、大”会吸引围观，不一定吸引有效客户。

### BM-3.6 账号矩阵必须角色互补

一个项目可以被不同账号讲，但角度必须不同：

| 账号 | 同一项目的角度 |
|---|---|
| 品牌主账号 | 市场位置、供应、成交和官方报告 |
| 马老师 | 哪类企业家家庭适合、决策与交易问题 |
| TONY | 建筑、空间、设计和跨城产品比较 |
| 小春哥 | 城市、板块和价格梯队 |
| 项目深耕经纪人 | 具体户型、在售库存、业主动机和带看规则 |

### BM-3.7 公域获客不等于成交能力

行业媒体的典型总结是：平台负责放大触达，最终成交仍依赖专业判断、私域沟通、房源和独立转化能力。[^social-industry-private]

因此马老师团队不得以“播放量高”替代以下能力：

- 线索辨识。
- 20 分钟诊断。
- 房源资源提纯。
- 长周期跟进。
- 谈判和后端交付。

### BM-3.8 把隐私保护变成可见服务

豪宅客户不希望隐私被用于炫耀。对外可公开的不是客户是谁，而是：

- 如何不在公开平台暴露楼栋和行程。
- 如何安排无打扰带看。
- 如何分级展示室内照片。
- 如何管理聊天、证件和资金资料。
- 如何让家庭成员分阶段参与。

### BM-3.9 公开“不会做什么”

可信度来自边界：

- 不为流量曝光未经授权的家。
- 不制造竞买。
- 不把每套房都称为天花板。
- 不用客户资产规模做人设。
- 不为佣金限制客户选择。

## BM-4. 不建议照搬的做法

1. **用数量掩盖定位不清**：高频发布只能放大已有定位，不能替代定位。
2. **只讲豪宅猎奇**：能带来围观，但容易形成“看热闹”而非“找顾问”的受众。
3. **过度使用急售、捡漏、天花板**：短期刺激点击，长期损伤证据可信度。
4. **把名人、圈层和合影当成交证明**：活动参与不等于服务能力。
5. **过度公开业主家与交易细节**：豪宅客户会用同行案例判断自己的隐私是否安全。
6. **复制同行话术和镜头**：没有独家资源和真实判断时，内容越像同行越难建立信任。
7. **只统计粉丝和点赞**：最重要的是有效咨询、业主委托、诊断、带看、成交和转介绍。
8. **用个人英雄主义替代团队交付**：顶豪交易需要法务、贷款、评估、验房、设计和物业协同。
