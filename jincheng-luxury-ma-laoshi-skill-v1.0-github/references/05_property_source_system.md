# 第十一部分｜房源资源系统

## 32. 房源来源分类

| 类型 | 说明 | 主要风险 |
|---|---|---|
| Developer Direct | 开发商直接资源 | 可售状态、价格口径、口头承诺、交付风险 |
| Owner Direct | 业主直接委托 | 权属、真实意愿、价格、抵押、共有、隐私 |
| Exclusive | 独家或深度委托 | 委托期限、价格合理性、业主配合 |
| Co-broker | 同业合作 | 信息失真、跳单、佣金、客户保护 |
| Off-market | 非公开或特殊房源 | 授权、真实性、传播边界、价格验证 |
| Distressed/Complex | 时间紧、抵押、诉讼或特殊结构 | 法律与资金风险，必须专业机构介入 |

## 33. 房源核验清单

### 新房

- 项目证照和可售状态。
- 房号、面积、价格和付款条件。
- 清水、精装或定制交付标准。
- 材料品牌、样板间差异和合同附件。
- 开发商主体、项目公司、交付和历史兑现。
- 车位、储藏室、会所、物业费和长期运营。
- 优惠、渠道佣金和有效期限。
- 退换、延期、变更和违约条款。

### 二手

- 产权人、共有关系和出售授权。
- 抵押、查封、租赁、户口、学位等状态。
- 业主动机、时间、心理价和替代方案。
- 税费、贷款、赎楼和付款结构。
- 装修、家具、设备、维修和瑕疵。
- 物业费、水电、车位和历史纠纷。
- 看房隐私、腾退和交付时间。

## 34. 房源资源分级

| 等级 | 标准 | 使用方式 |
|---|---|---|
| A | 真实、完整、可立即交易，关键资料已核验 | 可进入客户核心方案 |
| B | 基本真实，少量信息待补，但风险可控 | 可作为备选并标注缺口 |
| C | 房源可能存在，但授权、价格或状态不确定 | 仅内部跟踪，不向客户作确定表达 |
| D | 传闻、钓鱼、虚假、无授权或重大风险 | 禁止使用 |

## 35. 房源厚度指标

豪宅顾问的房源能力不能用“朋友圈发了多少套”衡量，而用以下指标：

- 目标板块有效房源覆盖率。
- A/B 级房源占比。
- 非公开和稀缺房源获取能力。
- 房源信息更新时间。
- 业主或开发商关键人的直接沟通深度。
- 客户需求出现后 48 小时内形成有效组合的能力。
- 新房、二手不偏科。

# 第十二部分｜一手与二手双引擎

## 36. 一手房：时机的生意

需要判断的不是“项目好不好”一个问题，而是：

- 什么时间进入。
- 哪一批次、楼栋、楼层和朝向。
- 价格和渠道政策是否处于窗口。
- 是否有特殊房源释放。
- 开发商兑现和交付是否可靠。
- 付款、定制、车位、精装是否可谈。
- 客户是否能在短时间完成家庭决策。

### 一手决策公式

```text
产品适配 × 房号稀缺 × 进入时机 × 交付确定性 × 条件可谈性
```

## 37. 二手房：时间的生意

二手豪宅的价格和成交由时间、业主动机、持有成本、替代方案和买方耐心共同决定。

必须跟踪：

- 首次挂牌时间和历次调价。
- 真实看房、报价和撤回记录。
- 业主为什么卖、何时必须卖。
- 是否依赖另一笔交易。
- 持有成本、贷款和家庭意见。
- 同类可替代房源和买家数量。
- 交易后端能否在业主时间内完成。

### 二手决策公式

```text
房源稀缺 × 业主动机 × 时间压力 × 交易可执行 × 买方耐心
```

## 38. 一二手中立原则

- 客户需求优先于佣金高低。
- 新房没有合适房源时必须打开二手。
- 二手无法满足交付、品质或时间时必须回到新房。
- 任何“只卖新房”或“只做二手”的路径依赖，都可能造成失单。
