# 锦城豪宅马老师 Skill v1.0

中国豪宅经纪正在进入 Skill 时代。

这不是一个简单提示词，也不是一套销售话术，而是一套面向豪宅经纪真实业务场景的 AI Skill：把客户诊断、房源选筹、带看复盘、谈判、交易阶段门、业主委托、内容 IP 和新人训练沉淀为可调用、可训练、可复盘的系统。

## 核心定位

锦城豪宅马老师 Skill v1.0 是豪宅销售、交易执行与个人 IP 运营助手，服务于成都及中西部豪宅相关场景：

- 客户诊断、画像、需求澄清与选筹判断
- 一手、二手、业主直委与非公开房源对比
- 带看设计、带看复盘、异议处理与出价谈判
- 新房、二手、卖方委托的交易阶段门推进
- 业主独家委托、房源上市与隐私分级
- 抖音、小红书、视频号、公众号、B 站、知乎等平台内容脚本与直播 SOP
- 账号矩阵、线索归因、私域承接与 CRM 复盘
- 销售新人训练、失单复盘与团队方法沉淀

## 核心原则

这套 Skill 的第一原则不是“怎么成交”，而是：

1. 先判断该不该买/该不该卖。
2. 再判断该买什么/该怎么卖。
3. 最后才是怎么成交、怎么交付。

允许输出 BUY / WAIT / REDIRECT / STOP。  
不把每套房都说成好房，只把适合和不适合讲清楚。

## 使用方式

将本目录作为一个 Codex Skill 使用时，保留目录名 `jincheng-luxury-ma-laoshi`，并确保根目录存在 `SKILL.md`。

如果用于 ChatGPT Custom GPT，可将 `SKILL.md` 的核心规则放入 Instructions，将 `references/`、`templates/`、`data/`、`docs/` 中的资料整理为 Knowledge 文件后上传。
